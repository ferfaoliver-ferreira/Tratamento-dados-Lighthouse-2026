# Tratamento-dados-Lighthouse-2026  

**Resolução de case da Indicium Academy**  

Participei de um desafio técnico intensivo da Indicium AI para o **Programa Light House 2026**, onde pude colocar em prática todo o ciclo de vida de dados — do tratamento de bases complexas até a implementação de modelos de Machine Learning.
Durante esse projeto, foquei em transformar dados brutos em decisões estratégicas, atuando em frentes como:   

✅ Análise Comportamental: Identificação de perfis de clientes de elite para otimização de campanhas.  

✅ Data Modeling: Estruturação de séries temporais robustas, corrigindo vieses analíticos com tabelas de calendário.   

✅ Inteligência Artificial: Desenvolvimento de previsões de demanda e sistemas de recomendação baseados em similaridade.

Além do desafio técnico com **Python, SQL e técnicas de Data Modeling**, o projeto reforçou minha capacidade de conectar o código com o impacto financeiro — um passo importante na minha transição para a Engenharia de Dados
